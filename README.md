Customer Churn Prediction & Analytics

Objective
Analyze telecom customer data to identify factors influencing customer churn and build a predictive model.

Tools & Technologies
Python
Pandas
NumPy
Seaborn
Matplotlib
Scikit-learn

Project Workflow
1. Data Cleaning and preprocessing
2. Exploratory Data Analysis
3. Customer segmentation analysis
4. Machine learning model development
5. Model evaluation

Key Insights
• Churn rate is approximately 26.5%
• Month-to-month contracts show the highest churn
• Customers with higher monthly charges are more likely to churn
• Fiber optic users have higher churn compared to DSL users

Model Performance
Logistic Regression accuracy: 99.92%
